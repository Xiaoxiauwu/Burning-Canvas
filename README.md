# Burning Canvas
本项目基本复现了 节目“《最强大脑 燃烧吧大脑》国际试炼赛 六维赛道第3轮” 中的游戏 “燃烧的画”。

UI基于ImGui。

设计了多种地图生成方式，以及三种游戏模式。

## 地图生成
你可以游玩随机生成的地图，也可以游玩由自己输入的图片生成的地图。

## 游戏模式
- 游玩本地地图的单机模式。
- 更新Rating的Rated模式。
- 与多个朋友联机PK的联机模式。

## 联机模式
联机模式下，你可以通过右键地图，将自己的地图分享给其他好友。
